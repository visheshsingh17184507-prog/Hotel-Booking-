# Hotel Booking

A Hotel Booking website built using HTML, CSS and JavaScript.

## Features
- Search Hotels
- Filter by Location
- Filter by Price
- Filter by Rating
- Sort Hotels
- Wishlist
- Hotel Details Page
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Fetch API

## How to Run
1. Download the project.
2. Open `index.html` in your browser.
